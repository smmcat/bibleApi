# bibleApi 接口文档

### 接口地址

```
http://127.0.0.1:85
```



### 参数接收

```
/bible
```

> GET 请求

| 参数名    | 值              | 说明             | 必填 |
| --------- | --------------- | ---------------- | ---- |
| shortName | string          | 圣经 书卷名 缩写 | 是   |
| chapter   | string / number | 对应该内容的 章  | 否   |
| node      | string / number | 对应该内容的 节  | 否   |

### 返回案例

若传入章节 ``chapter``  、``node``，则返回章节对应内容。若无 ``chapter``  、``node`` 则返回该 书卷 全文

```
/bible?shortName=hb&chapter=1
```
